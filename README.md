# Website User Behavior Analysis

## Objective
Analyze user behavior data to understand conversion patterns and identify opportunities to improve user experience and conversion rate.

## Dataset
E-commerce user behavior dataset including session activity, time on site, and purchase events.

## Key Insights
- Users with longer session duration are more likely to convert  
- Significant drop-off occurs before the purchase stage  
- Certain pages show high traffic but low conversion, indicating UX issues  

## Tools Used
- Python (pandas, matplotlib)  
- Power BI
